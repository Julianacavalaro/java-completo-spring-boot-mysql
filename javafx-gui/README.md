# JavaFX GUI Application

Aplicação desktop com interface gráfica usando JavaFX.

## Funcionalidades

- Formulário de cadastro de produtos
- Componentes visuais (TextField, Button, Label)
- Tratamento de eventos
- Validação de campos

## Executar

```bash
mvn javafx:run
```

## Requisitos

- Java 11 ou superior
- JavaFX 17.0.8 (incluído nas dependências)

## Componentes

A aplicação demonstra o uso de:
- `GridPane` - Layout em grade
- `TextField` - Campos de entrada
- `Button` - Botões de ação
- `Label` - Rótulos
- `Scene` - Cena principal

## Screenshot

A aplicação abre uma janela com um formulário contendo:
- Campo Nome
- Campo Preço
- Campo Quantidade
- Botão Cadastrar

Ao preencher os campos e clicar em "Cadastrar", uma mensagem de sucesso é exibida.
